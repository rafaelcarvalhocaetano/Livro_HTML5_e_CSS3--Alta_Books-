Example 12 — HiDPI images (Figure 9-32)

Adding HiDPI images using the width and height attributes

<!-- resizing a 640x480px image using the HTML width and height attributes -->
<img src="miss-baker.jpg" width="320" height="240" alt="Monkey Baker with a model Jupiter vehicle (NIX collection, NASA)">

Adding HiDPI images using CSS

/* resizing a 640x480px image using the CSS width and height properties */

<img class="hidpi" src="miss-baker.jpg" alt="Monkey Baker with a model Jupiter vehicle (NIX collection, NASA)">

CSS width and height overrule HTML attributes:

<img class="hidpi" src="img/miss-baker-640x480.jpg" alt="Monkey Baker with a model Jupiter vehicle (NIX collection, NASA)" width="640" height="480">
