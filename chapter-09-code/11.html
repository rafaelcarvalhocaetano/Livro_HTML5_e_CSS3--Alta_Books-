Example 11 — Introducing Media Queries

Using media queries when linking a stylesheet

<link rel="stylesheet" media="screen" href="screen.css">
<link rel="stylesheet" media="print" href="print.css">

…or inside a stylesheet

/* Non-media query CSS for any device (mobile-first) */
…
@media screen and (min-width: 24em) {
	body {
		/* CSS for everything except small devices… */
	}
	…
}
@media screen and (min-width: 42em) {
	body {
		/* CSS for tablets and larger screens… */
	}
	…
}

Targetting high-resolution screens now

@media screen and (-webkit-min-device-pixel-ratio: 2), screen and (min-resolution: 192dpi) {…}

Multiply by 96 to convert device-pixel-ratio into dpi (there are 96 CSS pixels per CSS inch)

…and in the future (once dppx is supported)

@media screen and (-webkit-min-device-pixel-ratio: 2), screen and (min-resolution: 2dppx) {…}

Example media queries

  • screen and (min-width: 534px)
  • screen and (max-width: 960px), screen and (max-height: 960px)
  • screen and (min-width: 20em) and (max-width: 32em)
  • screen and (max-width: 480px) and (-webkit-min-device-pixel-ratio: 2), screen and (max-width: 480px) and (min-resolution: 192dpi)

Inclusive media queries

@media all and (min-width: 534px) {…}
@media all and (min-width: 961px) {…}

Inclusive media queries

@media all and (min-width: 534px) and (max-width: 960px) {…}
@media all and (min-width: 961px) {…}

Windows mobile-compatible media queries

<!-- assuming a mobile-first design: -->
<!-- mobile styles for everyone -->
<link href="/css/base.css" media="all">

<!-- styles for wider viewports (invisible to IE8 and below) -->
<link href="/css/wide.css" media="all and (min-width: 30em)">

<!-- wider viewport styles for IE8 and below -->
<!--[if (lt IE 9)&(!IEMobile 7)]>
	<link rel="stylesheet" href="wide.css" media="all">
<![endif]-->

Viewport on mobile phones

<meta name="viewport" content="width=device-width, initial-scale=1.0">
