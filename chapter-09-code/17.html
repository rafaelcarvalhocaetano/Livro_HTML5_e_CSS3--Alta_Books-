Example 17 — CSS Generated Content for Paged Media Module

Adding chapter number and title-based running headers

h1 {
	string-set: chapter-title content-element;
	counter-increment: chapter;
}
title {
	string-set: book-title contents;
}
@page :left {
	@top-center {
		content: string(book-title);
	}
}
@page :right {
	@top-center {
		content: "Chapter " counter(chapter) ": " string(chapter-title);
	}
}

A paged user interface

@media paged {
	html {
		height: 100%;
		overflow: paged-x-controls;
	}
}
