Example — CSS Paged Media Module Level 3

Setting paper size, orientation, and page margins for left and right pages.

@page {
	size: A5 portrait;
}
/* channeling Müller-Brockmann */
@page :left {
	margin: 15mm 10mm 30mm 20mm;
}
@page :right {
	margin: 15mm 20mm 30mm 10mm;
}

Nesting inside @media rules:

@media print and (width: 210mm) and (height: 297mm) {
	@page {
		/* rules for A4 paper */
	}
}
@media print and (width: 8.5in) and (height: 11in) {
	@page {
		/* rules for US Letter paper */
	}
}
