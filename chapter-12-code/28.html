Example 28 — animation shorthand property (Figure 12-33)

Our last animation used individual animation-* properties:

:hover .box {
  animation-name: runner;
  animation-duration: 3s;
  animation-timing-function: ease-in-out;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}

It’s equivalent to this (much shorter) animation property:

:hover .box {animation: runner 3s ease-in-out infinite alternate;}

Remember we don’t have to include any properties with a default value — in this case animation-delay and animation-fill-mode

We can also specify more than one animation, using commas to separate values, for both individual animation-* properties and for the shorthand animation property. Both ways are demonstrated here:

In the individual properties form the first value of each property will be associated with the first animation-name value. If there aren’t enough values for the number of animation names, the values that are present are repeated, as is the case in the above example with animation-duration and animation-timing-function. By repeating the value(s) these will be treated as animation-duration: 3s, 3s 3s; and animation-timing-function: linear, steps(2,steps), linear;.
